Linux Ubuntu Notes

man "commanand" - will open a help page for any given command

Basic File Navagation:

cd - brings you to the home directory
cd name/ - will bring you to a specified directory
pwd - print working driectory - tells you what directory you are currently in
ls - list all files and directories in current directory
ls -l - list long form
ls -rtl - list long form but in reverse order (recently modified at the bottom)
ls -a - list all including hidden
. - will show hidden files and directories

Basic Text Files

echo "whatever you type" >foo.txt - will make a text file containing whatever you type
nano - will make a text file that you can interact with while in terminal
cat foo.txt - will print the context of foo.txt into the terminal
cat foo.txt bar.txt - will combine the two text files, order will match input order
touch "file name" - will create an empty text file with that name
diff foo.txt bar.txt - will print the differences between the text files in terminal
cp - copy - will copy a file to another directory
mv - move - will move a file to another directory, if no output file will be renamed
rv - remove - will delete a file, NO COMFIRMATION and CAN NOT BE UNDONE

Shortcut Keys

^ - CTRL
CTRL+ALT+T - opens terminal
^l - clears terminal
^d - closes terminal
^c - canels current command
^+TAB - switches between open windows
^e - goes to end of command line
^a - goes to start of command line
^u - deletes everything in command line
^s - will save file
TAB - auto fills commands where possible
TABTAB - will list all possible names for autofill

Wildcards

* - replaces zero or more characters ie. ls tes*.txt will show anthing that has tes....txt
? - replaces one character ie. tes?.txt will show test.txt, tesp.txt but not testing.txt
[] - will match any chacters in [] ie. test[].txt will show test1.txt test2.txt ect.

Updates

apt list -- upgradable - prints a list of things that can be updated sudo apt update - updates the package list for any updates (always run before upgrading) sudo apt upgrade - upgrades all outdated packages (may fail if sudo apt update was not ran)
